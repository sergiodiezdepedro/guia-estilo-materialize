Carrusel

El carrusel es compatible con gestos táctiles, por lo que es funcional para dispositivos móviles. Es necesario inicializarlo mediante código JS. Cualquier tipo de carrusel tiene la posibilidad de mostrar o no los indicadores, como una opción del script.

Carrusel Ampliado

El carrusel tiene una opción para convertirlo en uno más convencional, de ancho completo del contenedor. Tiene una inicialización JS distinta. Puede ser utilizado para mostrar otro tipo de contenidos, además de imágenes.

Acordeón

Estos elementos tienen varias opciones de personalización: efecto popout, con una sección abierta por defecto, efecto acordeón (al abrir un elemento se cierra el anterior), apertura total (se pueden abrir todos los elementos a la vez).

Dialogs

Son contenidos que no están visibles en la página, pero que se pueden mostrar mediante una acción del usuario con información adicional.

Toasts

Materialize provee una forma sencilla de enviar alertas discretas a sus usuarios mediante estos elementos. La forma más sencilla de implementarlos es añadir un evento onclick a un botón. Esta es la función

Materialize.toast(message, displayLength, className, completeCallback);

className (pasar clases como un parámetro opcional) y completeCallback (pasar un callback a la función toast que se ejecutará cuando este haya desaparecido) son opcionales.

Tooltips

Son pistas de texto pequeñas e interactivas principalmente para elementos gráficos. se añade la clase tooltipped al elemento y la posición top, bottom, left, right en el atributo data-tooltip para controlar la misma. Pensando en los dispositivos móbiles es aconsejable utilizar sólo las posiciones top y bottom.

Botones con dropdown

Se puede añadir una lista dropdown a cualquier botón. Es obligatorio que el atributo en data-activates sea idéntico a la id en la etiqueta ul. Se pueden incluir diivisores con la clase <li class="divider"></li>. Asimismo, se pueden usar iconos, dentro de la etiqueta a.

Es muy didáctico revisar las opciones del script dropdown-button en el archivo scripts.js.

Solapa de Novedades

Permite que se presente a los usuarios novedades o nuevas funcionalidades en momentos contextualemente relevantes. Debe ser usado con moderación y cuando sea oportuno. El texto no debe ser muy extenso.

La id del desencadenante y la clase data-activates del contenedor de la solapa deben ser iguales.

new_releases
Novedades Desopilantes

El desencadenante ha de tener un evento onclick para disparar la solapa.

Se puede personalizar el aspecto de la solapa, con las clases habituales, en el contenedor tap-target.

Media

Material Box

Es una implementación Material Design del conocido plugin Lightbox. Cuando un usuario hace clic en una imagen que puede ser ampliada, Material Box centra y amplía la imagen de una manera suave. Para salir del efecto, el usuario puede hacer clic en la imagen de nuevo, hacer scroll hacia abajo, o presionar la tecla ESC.

Para utilizar el recurso hay que añadir la clase materialboxed a la imagen.

Para incluir una pequeña descripción de la imagen, se añade con el atributo data-caption.

Slider de imágenes

Un carrusel de imágenes con transiciones, indicadores, autoplay. Es compatible con gestos táctiles, como el habitual scroll horizontal al deslizar el dedo.

Modales

Colores

La paleta de colores se basa en la de Material Design. Cada uno de estos colores está definido por una clase base de color y una clase opcional para oscurecerlo o aclararlo.

Color de fondo

Para aplicar un color de fondo, sólo hay que añadir el nombre del color y una clase para aclarar/oscurecer el elemento.

Este es un card-panel con las clases lime lighten-1. Además, tiene la clase hoverable para añadir un efecto 3-D cuando el puntero se sitúa sobre él.
Color de texto

Para aplicar un color al texto simplemente hay que añadir -text a la clase de color.

Este es un card-panel con la clase white-text para definir el color del texto.
Este es un card-panel con la clase blue-text text-darken-4 para fijar el color del texto con una variación más oscura del color de partida.
Este es un card-panel con la clase blue-text sin el modificador de tono.

Helpers

Alineación vertical

Se pueden centrar los ítems verticalmente añadiendo la clase valign-wrapper al contenedor que da cabida a los elementos que se quiere centrar.

Alineación horizontal

Hay tres clases para este propósito:

  • .left-align
  • .right-align
  • .center-align

Media

Se le pueden dar diferentes estilos a las imágenes con este framework.

Imágenes Responsive

Para conseguir que las imágenes sean responsive y se ajusten al ancho del contenedor, hay que añadir la clase responsive-img. Esto supone que tendrán las características CSS de max-width: 100% y height: auto.

Imágenes circulares

Para que las imágenes sean circulares, hay que añadir la clase circle en ellas.

Esta es una imagen responsive cuadrada. Añadiendo la clase circle se muestra circular.

La tarjeta tiene la clase z-depth-1 que define el tipo de sombra del elemento.

Vídeos

Hay un contenedor para incrustar vídeos (por ejemplo, de YouTube) que hace que sean responsive: video-container. También se dispone de una clase para que los vídeos HTML5 sean responsive, responsive-video.

Sombras

En Material Design, todo debe tener cierta profundidad en el eje z que determine lo cerca o lo lejos que está un elemento del fondo o de su contexto.

Se puede añadir un efecto de sombra a un elemento incluyendo la clase z-depth-x, donde X es un número entre 1 y 5. Se puede recurrir a la clase z-depth-0 para eliminar la sombra de un elemento que la tiene definida por defecto.

Tipografía

La fuente por defecto del framework es Roboto 2.0, una de las dos fuentes estándar de Material Design.

El framework se distribuye con la última versión de Roboto Google, en cinco tamaños de fuente diferentes: 100, 300, 400, 500, 600.

Flow Text

Un defecto común de muchos frameworks es la falta de soporte para el texto verdaderamente responsive. Mientras que otros elementos de la página responden fluidamente, el texto se redimensiona sobre una base fija.

Para mejorar este problema, para secciones cargadas de texto, se ha creado una clase que escala de manera fluida el tamaño del texto y la altura de línea, para optimizar la legibilidad del usuario. La longitud de línea se mantiene entre 45-80 caracteres y la altura de línea se escala para ser más grande en las pantallas más pequeñas.

Para ver el este Flow-Text en acción, hay que cambiar el tamaño del navegador lentamente y se observará el cambio del tamaño de este texto.

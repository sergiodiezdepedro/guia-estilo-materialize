Colores

La paleta de colores se basa en la de Material Design. Cada uno de estos colores está definido por una clase base de color y una clase opcional para aclararlo u oscurecerlo.

Color de fondo

Para aplicar un color de fondo, sólo hay que añadir el nombre del color y una clase para aclarar/oscurecer el elemento.

Este es un card-panel con las clases lime lighten-1. Además, tiene la clase hoverable para añadir un efecto 3-D cuando el puntero se sitúa sobre él.
Color de texto

Para aplicar un color al texto simplemente hay que añadir -text a la clase de color.

Este es un card-panel con la clase white-text para definir el color del texto.
Este es un card-panel con la clase blue-text text-darken-4 para fijar el color del texto con una variación más oscura del color de partida.
Este es un card-panel con la clase blue-text sin el modificador de tono.
